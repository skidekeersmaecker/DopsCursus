GIT bash here in folder

1. maak map en en ga naar met cd prompt naar url map.
2. npm init .
3. gegevens invullen
4. npm install express --save
5. cat package.json (kunnen we express zien)

6. mkdir client (onder deze map gaan we alle bestanden hosten op de webserver)
7. echo '<h1>Hallo</h1>' >> client/index.html
8. cat client/index.html
9. open client/index.html
10. start chrome client/index.html

11. vim server.js en in dit bestand de code (zie bestand)
12. naar folder gaan waar server.js in staat en: nodemon server.js
13. in browser: http://localhost:3000/ (als listen port = 3000)

14. npm install body-parser -- save
15. require en code in server.js schrijven (zie bestand)

16. met een app.post gebruiken we de extentie van chrome postman. Zet keuze op POST, vul in url en druk op send
